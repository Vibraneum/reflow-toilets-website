### Screen Reader Support
- Semantic HTML (section, h2, h3, h4, button)
- Alt text for timeline dots (decorative, empty alt)
- Link text is descriptive (not "click here")

## Animation Timing

### Accordion Expand/Collapse
```
Property:    max-height, padding
Duration:    0.3s
Easing:      ease
```

### Card Hover
```
Property:    transform, box-shadow
Duration:    0.3s
Easing:      ease
```

### Link Hover
```
Property:    color
Duration:    0.2s
Easing:      ease
```
