## Usage Guidelines

### DO:
✅ Use local images for better performance
✅ Match captions to actual image content
✅ Update alt text to describe what's shown
✅ Test builds after image changes
✅ Store new images in appropriate subdirectories

### DON'T:
❌ Fabricate captions based on assumptions
❌ Use external URLs when local images exist
❌ Reuse images for unrelated content
❌ Skip accessibility alt text
❌ Forget to optimize large images
