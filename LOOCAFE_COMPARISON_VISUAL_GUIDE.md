## Color Scheme

### Primary Colors
- **Brand Green**: `#34c759` - Used for headings, CTAs, highlights
- **Deep Green**: `#30b350` - Used in gradients
- **White**: `#ffffff` - Primary background
- **Light Gray**: `#f8f9fa` - Secondary background, subtle gradients

### Text Colors
- **Dark**: `#1a1a1a` - Primary headings and text
- **Gray**: `#6e6e73` - Secondary text, descriptions
- **Light Gray**: `#f1f1f1` - Borders, dividers

### Design Elements
- **Border Radius**: 12-24px for cards, 6-12px for buttons
- **Shadows**: Subtle `0 4px 20px rgba(0, 0, 0, 0.08)`
- **Hover Effect**: Lift `-4px` with deeper shadow
- **Transitions**: Smooth `0.3s ease`

## Typography

### Headings
- **Main Title**: 40-64px, 900 weight, -0.02em letter-spacing
- **Section Titles**: 28-32px, 800 weight
- **Card Titles**: 20-24px, 700 weight

### Body Text
- **Primary**: 16-18px, 400-600 weight, 1.6 line-height
- **Secondary**: 14-16px, 400 weight, 1.5 line-height
- **Labels**: 13-14px, 700 weight, uppercase, 0.1em letter-spacing

## Layout Sections

### 1. Header (100px padding)
- Centered alignment
- Max-width: 900px
- Label badge with green background
- Title + subtitle

### 2. Overview Cards (2-column grid)
- Max-width: 1000px
- 40px gap between cards
- White background with subtle shadow
- Hover effect: lift and deeper shadow

### 3. Comparison Table
- Full-width with max-width: 1400px
- Sticky header on scroll (optional)
- Alternating row hover effect
- Category column: 20% width
- Brand columns: 40% each
- Mobile: Horizontal scroll

### 4. Summary Cards (2-column grid)
- Green border (2px)
- Gradient background
- Checkmark list items
- Green checkmark icons

### 5. Partnership Section
- Full-width green gradient background
- Emoji icon (64px)
- White text
- CTA buttons with white background

## Responsive Breakpoints

### Desktop (1024px+)
- Full 2-column layout
- Table visible in full width
- All elements side-by-side

### Tablet (768px - 1023px)
- Table becomes scrollable horizontally
- Cards stack vertically
- Reduced padding

### Mobile (< 768px)
- Single column layout
- Table scrolls horizontally (800px min-width)
- Full-width CTA buttons
- Reduced font sizes

### Small Mobile (< 480px)
- Further reduced font sizes
- Minimal padding (16-24px)
- Optimized for one-handed use

## Accessibility Features

### Keyboard Navigation
- All links and buttons focusable
- Clear focus indicators
- Tab order follows visual order

### Screen Readers
- Semantic HTML (`<table>`, `<th>`, `<td>`)
- Descriptive headings hierarchy (H2, H3)
- Alt text for icons (using emoji, no images)
- ARIA labels where needed

### Color Contrast
- All text meets WCAG AA standards
- Green on white: 2.9:1 (AA Large)
- Dark on white: 16.1:1 (AAA)
- Gray on white: 5.2:1 (AA)

### Motion
- Smooth transitions (0.3s)
- Respects `prefers-reduced-motion`
- No auto-playing animations
